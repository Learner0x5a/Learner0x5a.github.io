Windows Host + Ubuntu Guest

Host配置

在资源管理器中的磁盘，例如D:\上右键-属性-共享-高级共享-勾选共享此文件夹，记一下网络路径，比如Desktop-jpjdlkr\d

其实可以对磁盘中具体的某个文件夹开启共享，但实测发现，当文件夹中的文件过大/过多的时候会失败（未知原因）
